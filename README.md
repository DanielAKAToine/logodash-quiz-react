# 🏎️ LogoDash - Automotive Quiz

An interactive challenge where you have to identify car brands from fragments of their logos.

## 🚀 Tecnologies   
* React.js
* CSS3 (Dark Theme)
* JavaScript (ES6+)

## 🧠 What I learn
* **State Management:** Score management and navigation between questions.
* **Randomisation Logic:** Implementation of a system to shuffle questions and select a random subset.
* **Conditional Rendering:** Switching between the game screen and the Game Over screen.
